```python
class Runtime(ABC):                          # HLD 3.8.1, 12дёӘжҠҪиұЎж–№жі•
    def load_tool() -> Any                   # еҠ иҪҪе·Ҙе…·

    def set_memory(key, value)               # и®ҫзҪ®и®°еҝҶ
    def resolve_import(path, from_file)      # и§ЈжһҗеҜје…Ҙ
```

### HelenHermesRuntime (е…·дҪ“е®һзҺ°)

- з»§жүҝ `Runtime` ABC
