# Helen Programming Agent — User Profile

## 编码风格

- 偏好函数式编程风格
- 使用 snake_case 命名
- 质量阈值：7 维评分 ≥ 7.5

## 工作流

- Git：commit + push 一步完成
- 测试：分批运行以避免 OOM
- 文档：中文叙述，英文代码/路径

## 偏好

- 优先使用 asyncio（内存受限环境）
- 文档必须可运行
- 自托管：核心用 Helen，Python 作为 shell

## 环境

- 系统：Linux (1.8GB RAM)
- Python: 3.11.15
- Helen 仓库: ~/helen/ (master 分支)
