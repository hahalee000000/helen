# Reports

会话过程中产生的各类报告文件集中存放在此目录。

## 约定

- **存放内容**：分析报告、评估报告、调研总结等会话产出物
- **不存放内容**：项目结构性文档（README、教程、API 文档等）
- **命名建议**：`<主题>.md` 或 `<日期>-<主题>.md`（如 `PERFORMANCE_ANALYSIS.md`）
