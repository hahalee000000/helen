## v1.6: 短期改进

### 1. 模块导入函数访问

**问题**: 导入模块后无法访问模块中定义的函数。

**解决方案**: 在 `ImportResolver` 中注册导入模块的函数到解释器的 `_functions` 字典。

**关键代码**:
```python
# helen/interpreter/interpreter.py
def visit_import_stmt(self, node: ImportStmtNode) -> object:
    # ... 解析模块 ...
    
    # 注册模块中的函数
    if hasattr(module, '_functions'):
        for name, func in module._functions.items():
            qualified_name = f"{alias}.{name}" if alias else name
            self._functions[qualified_name] = func
```

**测试**: `tests/language/test_v16_features.py`

## v1.7: 闭包与协议

### 1. 闭包/匿名函数

**语法**:
```helen
let add = fn(x, y) { return x + y }
print(add(1, 2))  // 3
```

**实现要点**:
1. **AST节点**: `LambdaNode` (继承 `ExpressionNode`)
2. **词法作用域**: 闭包捕获定义时的环境
3. **解释器**: `visit_lambda()` 创建 `Closure` 对象

**关键代码**:
```python
# helen/core/ast.py
@dataclass(frozen=True)
class LambdaNode(ExpressionNode):
    params: list[ParameterNode]
    body: list[StatementNode]
    return_type: TypeNode | None
    span: SourceSpan

# helen/interpreter/interpreter.py
def visit_lambda(self, node: LambdaNode) -> object:
    # 捕获当前环境
    return Closure(
        params=node.params,
        body=node.body,
        return_type=node.return_type,
        closure_env=self.environment,  # 词法作用域
        span=node.span
    )
```

**Pitfall**: 闭包必须捕获定义时的环境，而不是调用时的环境。

### 2. 协议声明与实现

**语法**:
```helen
protocol Printable {
    fn to_string(self): String
}

impl Printable for Point {
    fn to_string(self): String {
        return "Point(" + str(self.x) + ", " + str(self.y) + ")"
    }
}
```

**实现要点**:
1. **AST节点**: `ProtocolDeclNode`, `ImplDeclNode`
2. **语义分析**: 注册协议，检查实现是否匹配
3. **鸭子类型**: 不强制实现，运行时检查

**关键代码**:
```python
# helen/semantic/analyzer.py
def visit_protocol_decl(self, node: ProtocolDeclNode) -> None:
    # 注册协议
    self.protocols[node.name] = node

def visit_impl_decl(self, node: ImplDeclNode) -> None:
    # 检查协议是否存在
    if node.protocol_name not in self.protocols:
        self.errors.error(ErrorCode.UNDEFINED_PROTOCOL, ...)
    
    # 检查方法签名是否匹配
    protocol = self.protocols[node.protocol_name]
    for method in node.methods:
        self._check_method_signature(method, protocol)
```

**测试**: `tests/language/test_v17_protocols.py`

## v1.8: 函数式编程增强

### 1. 管道操作符 `|>`

**语法**:
```helen
let result = "hello" |> upper |> strip
// 等价于 strip(upper("hello"))
```

**实现要点**:
1. **Token**: `PIPE_RIGHT` (`|>`)
2. **优先级**: 低优先级（优先级 2），左结合
3. **AST节点**: `PipeExprNode`
4. **解释器**: 反糖化为函数调用

**关键代码**:
```python
# helen/core/tokens.py
PIPE_RIGHT = auto()  # |>

# helen/core/parser.py
# 在 Pratt 优先级表中添加
PIPE = 2  # 低优先级

# 注册中缀规则
self._rules[TokenType.PIPE_RIGHT].infix = self._pipe
self._rules[TokenType.PIPE_RIGHT].precedence = Precedence.PIPE

def _pipe(self, left: ExpressionNode) -> ExpressionNode:
    operator = self._previous()
    rule = self._rules.get(TokenType.PIPE_RIGHT, ParseFn())
    right = self._expression(rule.precedence + 1)
    return PipeExprNode(value=left, function=right, span=...)

# helen/interpreter/interpreter.py
def visit_pipe_expr(self, node: PipeExprNode) -> object:
    value = node.value.accept(self)
    func = node.function.accept(self)
    
    # 调用函数
    if callable(func):
        return func(value)
    # ... 处理其他情况
```

**Pitfall**: 管道操作符优先级必须低于赋值，否则 `let x = value |> fn` 会解析错误。

### 2. 模式匹配增强

#### 2.1 通配符模式

**语法**:
```helen
match value {
    case 1 { print("one") }
    case _ { print("other") }  // 匹配任何值
}
```

**实现要点**:
1. **Token**: `WILDCARD` (`_`)
2. **AST节点**: `WildcardPatternNode`
3. **解释器**: 总是匹配成功

**关键代码**:
```python
# helen/core/lexer.py
if lexeme == "_":
    tt = TokenType.WILDCARD

# helen/interpreter/interpreter.py
def visit_match_stmt(self, node: MatchStmtNode) -> object:
    for case in node.cases:
        if isinstance(case.pattern, WildcardPatternNode):
            matched = True
        # ...
```

#### 2.2 变量绑定模式

**语法**:
```helen
match value {
    case n if n > 0 { print("positive: " + str(n)) }
    case _ { print("non-positive") }
}
```

**实现要点**:
1. **AST节点**: `VariablePatternNode`
2. **语义分析**: 在 case 作用域中定义变量
3. **解释器**: 绑定值到变量名

**关键代码**:
```python
# helen/semantic/analyzer.py
def visit_case(self, node: CaseNode) -> None:
    self.symbols.enter_scope("match-case", "block")
    try:
        if isinstance(node.pattern, VariablePatternNode):
            sym = Symbol(name=node.pattern.name, kind="variable")
            self.symbols.define(node.pattern.name, sym)
        # 现在可以评估 guard 和 body
        if node.guard:
            node.guard.accept(self)
        self._visit_stmts(node.body)
    finally:
        self.symbols.exit_scope()

# helen/interpreter/interpreter.py
def visit_match_stmt(self, node: MatchStmtNode) -> object:
    for case in node.cases:
        if isinstance(case.pattern, VariablePatternNode):
            matched = True
            bindings = {case.pattern.name: subject}
        # ...
        if matched:
            self.environment = self.environment.enter_scope()
            for name, value in bindings.items():
                self.environment.define(name, value)
            # 执行 body
```

#### 2.3 类型模式

**语法**:
```helen
match value {
    case is String { print("it's a string") }
    case is Int { print("it's an int") }
    case _ { print("unknown") }
}
```

**实现要点**:
1. **关键字**: `is`
2. **AST节点**: `TypePatternNode`
3. **解释器**: 使用 `isinstance()` 检查类型

**关键代码**:
```python
# helen/core/tokens.py
IS = auto()

# helen/core/parser.py
def _case(self) -> CaseNode:
    if self._check(TokenType.IS):
        is_tok = self._advance()
        type_tok = self._consume(TokenType.IDENTIFIER, "Expected type name")
        type_name = type_tok.lexeme
        binding_name = None
        if self._check(TokenType.IDENTIFIER):
            binding_tok = self._advance()
            binding_name = binding_tok.lexeme
        pattern = TypePatternNode(type_name=type_name, binding_name=binding_name, span=...)
    # ...

# helen/interpreter/interpreter.py
def _check_type(self, value: object, type_name: str) -> bool:
    type_map = {
        "Int": int,
        "Float": float,
        "String": str,
        "Bool": bool,
        "List": list,
        "Map": dict,
        "Null": type(None),
    }
    expected_type = type_map.get(type_name)
    if expected_type is None:
        return False
    return isinstance(value, expected_type)
```

**支持的类型**: `String`, `Int`, `Float`, `Bool`, `List`, `Map`, `Null`
